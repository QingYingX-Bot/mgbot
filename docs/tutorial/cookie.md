若已绑定但提示 CK 失效，可发送 `#刷新ck`。  
未绑定或提示 SK 失效时，按以下步骤操作。

::: warning
请私聊机器人进行操作，避免账号信息泄露。
:::

1. 发送 `#扫码登录`，保存机器人发来的二维码图片  
   ![ck1](/tutorial/morehelp/bdbz/1.jpg)
2. 打开米游社 App，进入扫码界面  
   ![ck2](/tutorial/morehelp/bdbz/2.jpg)
3. 点击右上角“相册”，选择已保存二维码  
   ![ck3](/tutorial/morehelp/bdbz/3.jpg)
4. 确认登录完成绑定  
   ![ck4](/tutorial/morehelp/bdbz/4.jpg)
5. 绑定成功后机器人回复示例  
   ![ck5](/tutorial/morehelp/bdbz/5.jpg)
