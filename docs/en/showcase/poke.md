## Poke Ranking Plugin Guide

### Command Usage

#### Ranking Commands

- `#戳戳榜` / `#poke榜` - View the poke ranking in the current group
- `#戳戳榜10` / `#poke榜10` - View the top 10 poke ranking in the current group
- `#今日戳戳榜` / `#今日poke榜` - View today's poke ranking in the current group
- `#戳戳总榜` / `#poke总榜` - View the global poke ranking
- `#被戳戳榜` / `#被poke榜` - View the poked ranking in the current group
- `#今日被戳戳榜` / `#今日被poke榜` - View today's poked ranking in the current group
- `#被戳戳总榜` / `#被poke总榜` - View the global poked ranking

#### Personal Stats Commands

- `#查询戳戳` / `#查询poke` - Query your own poke stats
- `#查询戳戳 @用户` / `#查询poke @用户` - Query poke stats for a specified user
- `#查询戳戳 123456` / `#查询poke 123456` - Query poke stats for a specified QQ number
